# Waste-Management-Project
Overview

  A comprehensive, real-time solution for urban waste management designed to improve segregation, disposal tracking, and citizen engagement.
Features

  Live Dustbin Locator: A smart scheduling system integrated with Leaflet.js to track and locate bins in real-time, preventing overflows.

  Issue Tracking Platform: A dedicated interface for citizens to report waste-related problems directly to authorities.

  Gamified Disposal: Incorporates gamification elements to encourage and reward better waste disposal habits among users.

  Automated Scheduling: Optimizes garbage collection routes and timing for better city-wide efficiency.

Tech Stack

  Backend: Node.js, Express.js

  Frontend: EJS (Embedded JavaScript Templates)

  Database: MongoDB

  Maps/Geospatial: Leaflet.js

  DevOps: Docker

Installation

  Clone the Repository:
  
  git clone https://github.com/your-username/waste-management-system.git

Install Dependencies:
  npm install

  Run with Docker:
  docker-compose up

Usage

  Open your browser and navigate to http://localhost:3000.

    Use the Live Map to find the nearest available dustbin.

    Report any waste issues through the Issue Tracking dashboard.
